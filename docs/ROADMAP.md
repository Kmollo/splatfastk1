# Roadmap

## MVP

- `splatforge doctor`
- `splatforge create video.mp4`
- Frame extraction with FFmpeg
- COLMAP/GLOMAP reconstruction orchestration
- Brush backend command planning
- Local tool detection for SkySplat-style Brush binaries
- Local upload UI with job status
- Project manifest and output folder layout
- Blender add-on starter

## Next

- Verify Brush command arguments against installed release binaries.
- Run a full local video-to-PLY test scene.
- Add a browser-based Gaussian splat preview for generated PLY files.
- Add footage quality scoring.
- Add preview thumbnails and reconstruction summary.
- Auto-detect installed Blender splat importers.
- Package a simple desktop UI.

## Later

- Cloud GPU job runner.
- Team/project sharing.
- Batch processing.
- Native Blender panel with progress updates.
